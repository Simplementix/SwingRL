## Critical Integration Gaps (Blockers)

### INT-01: main.py ExecutionPipeline TypeError (CRITICAL)

**Location:** `scripts/main.py:231`
**Issue:** `ExecutionPipeline(config=config, db=db)` passes only 2 of 5 required arguments. Missing: `feature_pipeline`, `alerter`, `models_dir`.
**Impact:** Production entrypoint cannot start. Trading cycles will never execute via scheduler.
**Fix:** Create `FeaturePipeline` and `Alerter` instances before `ExecutionPipeline`, pass all 5 args.
**Affected:** PAPER-01, PAPER-02, PAPER-12

### INT-02: Feature Tables Not in init_db.py (CRITICAL)

**Location:** `src/swingrl/features/schema.py` → `src/swingrl/data/db.py`
**Issue:** `init_feature_schema()` is only called from `compute_features.py`, not from `init_db.py` or `DatabaseManager.init_schema()`. Feature tables don't exist until `compute_features.py` runs.
**Impact:** FeaturePipeline fails on first query if `compute_features.py` wasn't run first.
**Fix:** Call `init_feature_schema(conn)` from `DatabaseManager._init_duckdb_schema()`.
**Affected:** FEAT-11, DATA-09

### INT-03: stop_polling.py Wrong Table Name (HIGH)

**Location:** `src/swingrl/scheduler/stop_polling.py:82`
**Issue:** Queries `position_tracker` table; actual table is `positions`. Also references non-existent columns `stop_loss_price` and `take_profit_price`.
**Impact:** Crypto stop-price polling silently fails every 60 seconds.
**Fix:** Update table name to `positions`, handle stop/TP columns.
**Affected:** PAPER-10

### INT-04: Emergency Trigger Tables Missing (HIGH)

**Location:** `src/swingrl/execution/emergency.py:434-476`
**Issue:** Queries `market_indicators`, `inference_log`, `api_status_log` — none exist in schema.
**Impact:** All 3 automated emergency triggers silently fail. Manual `emergency_stop.py` still works.
**Fix:** Create tables in schema or rewrite queries against existing tables.
**Affected:** PROD-07

### INT-05: Wrong FRED Import Path (MEDIUM)

**Location:** `src/swingrl/scheduler/jobs.py:304,328`
**Issue:** Imports from `swingrl.data.ingestors.fred` — no `ingestors/` package exists. Actual path: `swingrl.data.fred`.
**Impact:** Weekly fundamentals and monthly macro refresh jobs silently fail.
**Fix:** Update import path.
**Affected:** DATA-04, FEAT-04
