    **Step 1 — Create `services/memory/routers/live.py`:**

    Follow the pattern from `services/memory/routers/training.py`:

    ```python
    """Live trading advisor endpoints: 5 endpoints for cycle-level memory enrichment.

    All endpoints require X-API-Key header.
    Called by LiveMemoryClient in src/swingrl/memory/live_client.py.
    """
    from __future__ import annotations

    import time
    from typing import Any

    import structlog
    from auth import verify_api_key
    from db import insert_live_decision_async
    from fastapi import APIRouter, Depends
    from memory_agents.live_agent import LiveAgent
    from pydantic import BaseModel

    log = structlog.get_logger(__name__)
    router = APIRouter()
    _AGENT = LiveAgent()
    ```

    **Request models** (shared base):
    ```python
    class LiveRequest(BaseModel):
        env: str  # "equity" or "crypto"
        context: dict[str, Any]  # Market state, regime, portfolio metrics
    ```

    **Response models** (one per endpoint):
    ```python
    class CycleGateResponse(BaseModel):
        approved: bool
        confidence: float
        rationale: str
        latency_ms: float

    class BlendWeightsResponse(BaseModel):
        deltas: dict[str, float]  # {PPO: float, A2C: float, SAC: float}
        confidence: float
        rationale: str
        latency_ms: float

    class RiskThresholdsResponse(BaseModel):
        max_dd: float | None
        daily_limit: float | None
        confidence: float
        rationale: str
        latency_ms: float

    class PositionAdviceResponse(BaseModel):
        kelly_scale_factor: float
        hold_bias: bool
        max_simultaneous_positions: int | None
        confidence: float
        rationale: str
        latency_ms: float

    class TradeVetoResponse(BaseModel):
        vetoed: bool
        confidence: float
        rationale: str
        latency_ms: float
    ```

    **Endpoint pattern** (same for all 5):
    ```python
    @router.post("/live/cycle_gate", response_model=CycleGateResponse)
    async def cycle_gate(body: LiveRequest, _key: str = Depends(verify_api_key)):
        t0 = time.monotonic()
        result = await _AGENT.query_cycle_gate(body.env, body.context)
        latency_ms = (time.monotonic() - t0) * 1000
        # Log decision to memory_live_decisions table
        await insert_live_decision_async(
            env=body.env, endpoint="cycle_gate",
            decision_json=json.dumps(result), rationale=result.get("rationale", ""),
            confidence=result.get("confidence", 0.0), latency_ms=latency_ms,
            clamped=0,
        )
        return CycleGateResponse(
            approved=result.get("approved", True),
            confidence=result.get("confidence", 0.0),
            rationale=result.get("rationale", "safe_default"),
            latency_ms=latency_ms,
        )
    ```

    Repeat for all 5 endpoints. Each includes the decision logging step.

    For blend_weights, add clamped flag detection: if any delta was modified by clamping, set `clamped=1`.
