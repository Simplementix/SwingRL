    ```python
    def apply_overrides(self, max_dd: float | None, daily_limit: float | None) -> None:
        """Save current thresholds and apply memory-advised overrides (tighten only)."""
        self._saved_max_dd = self._max_dd
        self._saved_daily_limit = self._daily_limit
        if max_dd is not None:
            self._max_dd = min(self._max_dd, max_dd)  # tighten only
        if daily_limit is not None:
            self._daily_limit = min(self._daily_limit, daily_limit)  # tighten only
        log.info("cb_overrides_applied",
                 env=self._environment,
                 original_max_dd=self._saved_max_dd, new_max_dd=self._max_dd,
                 original_daily_limit=self._saved_daily_limit, new_daily_limit=self._daily_limit)

    def reset_overrides(self) -> None:
        """Restore original thresholds (call at start of each cycle)."""
        if hasattr(self, "_saved_max_dd"):
            self._max_dd = self._saved_max_dd
            self._daily_limit = self._saved_daily_limit
            log.debug("cb_overrides_reset", env=self._environment,
                      max_dd=self._max_dd, daily_limit=self._daily_limit)
    ```

    ```python
    """Pre-scheduler startup validation. Called by main.py before scheduler init.

    Hard fail (sys.exit(1)) on critical checks.
    Warn (log.warning) on advisory checks.
    """
    from __future__ import annotations

    import os
    import sys
    from pathlib import Path

    import structlog

    log = structlog.get_logger(__name__)
    ```

    **Hard-fail checks (exit 1 on failure):**

    1. `check_env_keys()` — verify all required env vars are present and non-empty:
       - Required: `ALPACA_API_KEY`, `ALPACA_SECRET_KEY`, `BINANCE_API_KEY`, `BINANCE_SECRET_KEY`, `FRED_API_KEY`
       - Log each missing key. Exit 1 if any missing.

    2. `check_broker_credentials()` — ping Alpaca paper API:
       ```python
       from alpaca.trading.client import TradingClient
       client = TradingClient(api_key=api_key, secret_key=secret_key, paper=True)
       client.get_account()
       ```
       Exit 1 on any exception. Do NOT ping Binance.US in paper mode (BinanceSimAdapter doesn't need live keys).

    3. `check_db_tables()` — verify SQLite and DuckDB:
       - SQLite: connect to `config.system.sqlite_path`, `SELECT name FROM sqlite_master WHERE type='table'`, check for `trades`, `positions`, `portfolio_snapshots`, `circuit_breaker_events`
       - DuckDB: connect to `config.system.duckdb_path`, `SELECT table_name FROM information_schema.tables`, check for `ohlcv_daily`, `ohlcv_4h`, `features_equity`, `features_crypto`
       - Exit 1 if connection fails or critical tables missing

    4. `check_bind_mount_writability()` — for each of `data/`, `db/`, `models/`, `logs/`, `status/`:
       - `Path(d) / ".startup_check_tmp"` touch and delete
       - Exit 1 if write fails (PermissionError)

    **Warn-only checks:**

    5. `check_model_files()` — for each env in `["equity", "crypto"]` and algo in `["ppo", "a2c", "sac"]`:
       - Check `models/active/{env}/{algo}/model.zip` exists
       - Log warning per missing file, do NOT exit

    6. `check_memory_service()` — HTTP GET to `{memory_agent.base_url}/health`:
       - urllib.request.urlopen with 3s timeout
       - Log warning on any failure, do NOT exit

    7. `check_memory_seeding()` — HTTP GET to `{memory_agent.base_url}/debug/memories`:
       - Parse JSON, check `count` field
       - If count == 0, log.warning("MEMORY NOT SEEDED — run seed_memory_from_backtest.py before first live cycle")
       - Do NOT exit

    8. `check_feature_dimensions()` — load one observation per env:
       - Expected: 164 equity, 47 crypto (from CONTEXT.md training constants)
       - Log warning if mismatch, do NOT exit

    ```python
    def run_startup_checks(config: SwingRLConfig) -> bool:
        """Run all startup checks. Returns True if all hard checks pass."""
        log.info("startup_checks_begin")
        # Hard checks — each calls sys.exit(1) on failure
        check_env_keys()
        check_broker_credentials()

        check_memory_service(config)
        check_memory_seeding(config)
        check_feature_dimensions(config)
        log.info("startup_checks_complete")
        return True
    ```
