    2. `run_equity(config: SwingRLConfig, backfill: bool) -> int`:
       - Count rows before in `ohlcv_daily`
       - Create `AlpacaIngestor(config)`
       - Call `ingestor.run_all(config.equity.symbols, since=None if backfill else "incremental")`
       - If failed symbols returned, raise `DataError`
       - Count rows after, return delta
       - Log: `log.info("equity_ingestion_complete", rows_added=delta)`

    3. `run_crypto(config: SwingRLConfig, backfill: bool) -> int`:
       - Count rows before in `ohlcv_4h`
       - Create `BinanceIngestor(config)`
       - If backfill: call `ingestor.backfill(symbol)` for each symbol in `config.crypto.symbols`
       - If not backfill: call `ingestor.run_all(config.crypto.symbols, since=None)`, raise DataError on failures
       - Count rows after, return delta
       - Log result

    4. `resolve_crypto_gaps(df: pd.DataFrame, symbol: str) -> pd.DataFrame`:
       - Forward-fill with `limit=2`: `filled = df.ffill(limit=2)`
       - Check for remaining NaN: `remaining = filled.isna().any(axis=1).sum()`
       - If remaining > 0: `log.error("crypto_gap_unfillable", ...)` and raise `DataError`
       - Return filled DataFrame

    5. `run_macro(config: SwingRLConfig, backfill: bool) -> int`:
       - Count rows before in `macro_features`
       - Create `FREDIngestor(config)`
       - Call `ingestor.run_all(backfill=backfill)`
       - If failed series, raise `DataError`
       - Count rows after, return delta

    6. `run_features(config: SwingRLConfig) -> None`:
       - Create `DatabaseManager(config)`, open `duckdb()` cursor
       - Create `FeaturePipeline(config, conn)`
       - Call `pipeline.compute_equity()` then `pipeline.compute_crypto()`
       - Log: `log.info("feature_computation_complete")`

    7. `run_pipeline(config: SwingRLConfig, backfill: bool) -> int`:
       - Run stages in order: equity -> crypto -> macro
       - Sum row deltas
       - If rows_added > 0: run features; else log skip
       - Run verification (always): `result = run_verification(config)`
       - Write report: `write_report(result, _VERIFICATION_PATH)`
       - Print summary: `print_summary(result)`
       - Return 0 if result.passed else 1

    8. `_parse_args(argv: list[str] | None = None) -> argparse.Namespace`:
       - `--backfill` flag (store_true): full historical fetch
       - `--config` option: path to config YAML (default: `config/swingrl.yaml`)

    9. `main(argv: list[str] | None = None) -> None`:
       - Parse args
       - `configure_logging()`
       - `config = load_config(args.config)`
       - `sys.exit(run_pipeline(config, args.backfill))`

    10. `if __name__ == "__main__": main()` block for `python -m` invocation

    Constants:
    - `_VERIFICATION_PATH = Path("data/verification.json")`
