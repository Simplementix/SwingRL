       ```
       Step 3: Locate latest backups (ls backups/sqlite/, backups/duckdb/)

       Step 8: Start container (docker compose up -d)
       Step 9: Verify first cycle (wait for scheduler, check healthcheck passes)
       ```
       Each step prints PASS/FAIL with details. Script can be run in `--dry-run` mode (validates steps 3-7 without destructive operations) or `--full` mode (executes all 9 steps).
