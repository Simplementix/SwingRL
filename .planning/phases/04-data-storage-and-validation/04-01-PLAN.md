    **DatabaseManager (src/swingrl/data/db.py):**
    - Singleton via threading.Lock in __new__, with _initialized guard in __init__
    - `reset()` classmethod that sets `_instance = None` and closes connections (for test isolation)
    - `__init__(self, config: SwingRLConfig)` — extract paths from `config.system.duckdb_path` and `config.system.sqlite_path`
    - `duckdb()` context manager — lazy-init persistent DuckDB connection, yield cursor() for thread safety, close cursor on exit
    - `sqlite()` context manager — create new connection each call (SQLite is lightweight), set WAL mode + foreign keys + row_factory, auto-commit/rollback
    - `_write_lock` (threading.Lock) used only in init_schema() for DDL safety
    - `init_schema()` — CREATE TABLE IF NOT EXISTS for all 5 DuckDB tables + all 10 SQLite tables + indexes + views. Use write lock. Also INSTALL sqlite; LOAD sqlite for sqlite_scanner.
    - `attach_sqlite(conn)` — ATTACH SQLite DB as 'ops' with TYPE sqlite
    - `close()` — graceful shutdown, close DuckDB persistent connection

    **DuckDB DDL (inside init_schema):**
    - ohlcv_daily: symbol TEXT, date DATE, open DOUBLE, high DOUBLE, low DOUBLE, close DOUBLE, volume BIGINT, adjusted_close DOUBLE, fetched_at TIMESTAMP DEFAULT current_timestamp, PRIMARY KEY (symbol, date)
    - ohlcv_4h: symbol TEXT, datetime TIMESTAMP, open DOUBLE, high DOUBLE, low DOUBLE, close DOUBLE, volume DOUBLE, source TEXT, fetched_at TIMESTAMP DEFAULT current_timestamp, PRIMARY KEY (symbol, datetime)
    - macro_features: date DATE, series_id TEXT, value DOUBLE, release_date DATE, PRIMARY KEY (date, series_id)
    - data_quarantine: quarantined_at TIMESTAMP DEFAULT current_timestamp, source TEXT, symbol TEXT, raw_data_json TEXT, failure_reason TEXT, severity TEXT
    - data_ingestion_log: run_id TEXT PRIMARY KEY, timestamp TIMESTAMP DEFAULT current_timestamp, environment TEXT NOT NULL, symbol TEXT NOT NULL, status TEXT NOT NULL, rows_inserted INTEGER DEFAULT 0, errors_count INTEGER DEFAULT 0, duration_ms INTEGER, binance_weight_used INTEGER
