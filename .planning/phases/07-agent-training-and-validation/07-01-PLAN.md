  <behavior>
    - annualized_sharpe: returns (mean/std)*sqrt(periods_per_year). Returns NaN for len<2 or zero-std. Equity periods_per_year=252, crypto=2191.5
    - sortino_ratio: uses downside deviation (only negative returns) instead of full std. Returns NaN for no downside dev
    - calmar_ratio: annualized_return / abs(max_drawdown). Returns NaN if MDD is zero
    - rachev_ratio: CVaR(5%) of gains / CVaR(5%) of losses (absolute). Returns NaN if no gains or no losses
    - max_drawdown: worst peak-to-trough decline as a positive fraction (0.15 = 15% drawdown)
    - avg_drawdown: mean of all drawdown values in the underwater curve
    - max_drawdown_duration: longest number of bars spent in drawdown before recovery
    - compute_trade_metrics(trades): returns dict with win_rate, profit_factor, total_trades, avg_win, avg_loss, trade_frequency_per_week. trades is list of dicts with pnl float. PF = sum(wins)/abs(sum(losses)), returns inf if no losses
  </behavior>

    2. `sortino_ratio(returns: np.ndarray, periods_per_year: float, risk_free_rate: float = 0.0) -> float` -- uses only negative returns for denominator

    4. `rachev_ratio(returns: np.ndarray, alpha: float = 0.05) -> float` -- CVaR of top alpha gains / CVaR of bottom alpha losses

    6. `avg_drawdown(returns: np.ndarray) -> float` -- mean of underwater curve
    7. `max_drawdown_duration(returns: np.ndarray) -> int` -- bars in longest drawdown
    8. `compute_trade_metrics(trades: list[dict[str, Any]], total_bars: int, bars_per_week: float) -> dict[str, float]` -- win_rate, profit_factor, total_trades, avg_win, avg_loss, trade_frequency_per_week
