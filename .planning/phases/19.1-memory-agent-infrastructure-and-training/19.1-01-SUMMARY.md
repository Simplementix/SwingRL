## Decisions Made

- Self-contained service boundary: zero imports from src/swingrl/ to avoid cross-container ImportError (Pitfall 4)
- consolidation_quality table in SQLite memory.db (not DuckDB) — operational data belongs with what it logs
- Conflict detection via keyword overlap heuristic — sufficient for phase, avoids embedding dependency
- Ollama healthcheck uses `wget` not `curl` — curl absent from ollama/ollama image
- QueryAgent clamp bounds defined inline — must be manually kept in sync with bounds.py
- `fastapi`, `uvicorn`, `apscheduler` installed into main project venv for running unit tests locally (Rule 3 deviation)
