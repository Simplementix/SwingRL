    **2. Create src/swingrl/envs/portfolio.py:**
    `PortfolioSimulator` class with:
    - `__init__(self, initial_cash: float, n_assets: int, transaction_cost_pct: float)` -- sets cash, zero shares array, empty trade_log
    - `rebalance(self, target_weights: np.ndarray, prices: np.ndarray) -> float` -- compute current total value, determine target dollar values per asset, compute delta, deduct cost = sum(abs(delta)) * cost_pct, update shares = target_values / prices (use np.where(prices > 0, prices, 1.0) for safety), update cash = total_value - sum(target_values) - cost, log trades, return cost
    - `portfolio_value(self, prices: np.ndarray) -> float` -- cash + np.sum(shares * prices)
    - `asset_weights(self, prices: np.ndarray) -> np.ndarray` -- per-asset value / total_value
    - `reset(self, initial_cash: float) -> None` -- restore to initial state
    - `trade_log: list[dict]` -- records of all trades

    Also add a standalone function `process_actions(raw_actions: np.ndarray, current_weights: np.ndarray, deadzone: float) -> np.ndarray` that:
    1. Applies numerically stable softmax: `exp(x - max(x)) / sum(exp(x - max(x)))`
    2. Applies deadzone: where abs(softmax_weight - current_weight) < deadzone, keep current_weight
    3. Returns target weights array

    **3. Create src/swingrl/envs/rewards.py:**

    - `compute(self, daily_return: float) -> float` -- append return, if < 2 returns return 0.0, compute mean/std (ddof=1), if std < 1e-8 return 0.0, return mean/std
    - `reset(self) -> None` -- clear deque
