    1. Create tests/data/fixtures/fred_cpiaucsl_releases.json — mock return value of fred.get_series_all_releases("CPIAUCSL"). Structure: list of {date, realtime_start, value} entries. ~20 entries showing CPI values with vintage dates (some revised). Serialize dates as ISO strings.

    2. Create tests/data/fixtures/fred_vixcls_series.json — mock return value of fred.get_series("VIXCLS"). Structure: {date: value} mapping. ~20 entries of daily VIX values.

    3. Create tests/data/test_fred_ingestor.py — mock fredapi.Fred using unittest.mock.patch (fredapi uses HTTP internally but the Fred class is the interface to mock). Use monkeypatch for FRED_API_KEY env var. Each test docstring references DATA-04.

    4. Create src/swingrl/data/fred.py:
       - Module-level constants:
         DAILY_SERIES = ["VIXCLS", "T10Y2Y", "DFF"] (no vintage tracking needed)
         VINTAGE_SERIES = ["CPIAUCSL", "UNRATE"] (use get_series_all_releases)
         ALL_SERIES = DAILY_SERIES + VINTAGE_SERIES
       - FREDIngestor(BaseIngestor): constructor reads FRED_API_KEY from os.environ, creates Fred(api_key=...)
       - fetch(series_id, since):
         If series_id in VINTAGE_SERIES: call fred.get_series_all_releases(series_id). Rename realtime_start -> vintage_date. Set index to observation_date (date column).
         If series_id in DAILY_SERIES: call fred.get_series(series_id, observation_start=start_date). Convert to DataFrame with value column. Set index to observation_date.
         start_date = "2016-01-01" for backfill, or max(observation_date) + 1 day for incremental.
       - validate(df, series_id): delegates to DataValidator(source="fred"). FRED validation: null values are expected (NaN for missing days like weekends for VIX), so row-level null check is lenient — only quarantine if value is NaN on a day when the series should have published.
       - store(df, series_id): ParquetStore.upsert(data_dir / "macro" / f"{series_id}.parquet", df)
       - run_all(series_ids, since): loop with try/except per series, return failed list
       - Retry: 3 attempts with time.sleep(2**attempt) on exceptions
       - if __name__ == "__main__" block with argparse: --series (nargs="+", default=ALL_SERIES), --backfill (store_true). sys.exit(1) if any failures.

    5. Update src/swingrl/data/__init__.py to export FREDIngestor.

    Note: FRED data validation is different from OHLCV — there's no OHLC ordering, no volume. The DataValidator for source="fred" should only apply: null value check (step 1), duplicate date check (step 8), gap detection per-series cadence (step 9), stale data check (step 10).
