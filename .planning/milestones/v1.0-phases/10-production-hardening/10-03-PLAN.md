    - Test: archive(active_path) moves model from models/active/ to models/archive/
    - Test: delete(archive_path) removes model from models/archive/
    - Test: Invalid transition (e.g., TRAINING -> ACTIVE skipping SHADOW) raises ModelError
    - Test: promote with no existing active model works (no archive step needed)
    - Test: smoke_test_model validates deserialization succeeds
    - Test: smoke_test_model detects degenerate actions (all identical outputs)
    - Test: smoke_test_model rejects model with NaN outputs
    - Test: smoke_test_model checks inference time < 100ms
  </behavior>
  <action>
    1. Create src/swingrl/shadow/__init__.py (empty).
    2. Create tests/shadow/__init__.py (empty).

    3. Create src/swingrl/shadow/lifecycle.py:
       - `ModelState` enum: TRAINING, SHADOW, ACTIVE, ARCHIVE, DELETED
       - `ModelLifecycle` class:
         - `__init__(self, models_dir: Path)`: stores base models_dir, creates active/shadow/archive subdirs if missing
         - `deploy_to_shadow(self, model_path: Path, env_name: str) -> Path`: Copies model file to models/shadow/{env_name}/. Returns destination path. Logs via structlog.
         - `promote(self, env_name: str) -> Path`: Moves current active model (if any) to models/archive/{env_name}/ with timestamp suffix. Moves shadow model to models/active/{env_name}/. Returns new active path. Raises ModelError if no shadow model exists.
         - `archive(self, env_name: str) -> Path`: Moves active model to archive. Returns archive path.
         - `rollback(self, env_name: str) -> Path`: Finds most recent archive model, moves it to active. Returns restored path. Raises ModelError if no archive exists.
         - `delete_archived(self, model_path: Path) -> None`: Removes a specific archived model file.
         - `get_state(self, env_name: str) -> dict`: Returns dict with active_model, shadow_model, archive_count per environment.

       - `smoke_test_model(model_path: Path, env_name: str, obs_dim: int) -> dict[str, bool]`:
         Standalone function performing 6-point smoke test:
         1. Deserialization: Load model via SB3 load (PPO/A2C/SAC detected from filename)
         2. Output shape: predict() returns action array of correct shape
         3. Non-degenerate: Run 10 predictions with different random obs, verify actions are not all identical
         4. Inference speed: Single predict() < 100ms
         5. VecNormalize: If .pkl file exists alongside model, verify it loads
         6. No NaN: All 10 predictions contain no NaN values
         Returns dict of check_name -> pass/fail bool.

    4. Create tests/shadow/test_lifecycle.py with lifecycle transition tests using tmp_path for models_dir. Create dummy model files (just empty .zip files for file-move tests).

    5. Create tests/test_deploy_smoke.py testing smoke_test_model with mocked SB3 model (mock predict() to return valid/degenerate/NaN arrays). Test inference timing with mock.
  </action>

  </verify>
  <done>ModelLifecycle manages Training->Shadow->Active->Archive->Delete transitions, smoke test validates 6 checks, all tests pass</done>
</task>

       where obs_dim is 156 for equity, 45 for crypto
    f. Print SUCCESS or FAILURE with details
    g. Exit 0 on success, 1 on failure
