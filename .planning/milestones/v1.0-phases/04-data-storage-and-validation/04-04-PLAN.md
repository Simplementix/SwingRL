    ```python
    class CorporateActionDetector:
        """Detect and record corporate actions (splits, dividends)."""

        def __init__(self, db: DatabaseManager) -> None: ...

        def detect_overnight_spike(
            self, symbol: str, prev_close: float, curr_close: float,
            date: str, environment: Literal["equity", "crypto"] = "equity",
        ) -> bool: ...

        def record_action(
            self, symbol: str, action_type: str, effective_date: str,
            ratio: float | None = None, amount: float | None = None,
        ) -> None: ...

        def is_known_action(self, symbol: str, date: str) -> bool: ...

        def check_and_suppress(
            self, symbol: str, date: str, pct_change: float,
            environment: Literal["equity", "crypto"] = "equity",
        ) -> bool: ...
    ```

    Implementation:
    - `detect_overnight_spike()`: Calculate abs(pct_change), compare to threshold (30% equity, 40% crypto). If over threshold AND not already in corporate_actions table, return True (potential unrecorded action).
    - `record_action()`: INSERT INTO corporate_actions with uuid action_id, mark processed=0
    - `is_known_action()`: SELECT from corporate_actions WHERE symbol=? AND effective_date=?
    - `check_and_suppress()`: If overnight spike detected but is_known_action returns True, return True (suppress quarantine). Used by DataValidator.
