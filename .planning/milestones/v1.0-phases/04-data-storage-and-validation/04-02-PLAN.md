    **Modify run() to add DuckDB sync and logging:**
    ```
    run(symbol, since=None):
      run_id = str(uuid.uuid4())
      start = time.perf_counter()
      rows_inserted = 0
      errors_count = 0
      status = "success"
      try:
        raw = self.fetch(symbol, since)
        if raw.empty:
          status = "no_data"
          return
        clean, quarantine = self.validate(raw, symbol)
        errors_count = len(quarantine)
        if not quarantine.empty:
          self._store_quarantine(quarantine, symbol)
        if not clean.empty:
          self.store(clean, symbol)  # Parquet (unchanged)
          rows_inserted = self._sync_to_duckdb(clean, symbol)
      except Exception as e:
        status = "failed"
        log.error("ingestor_run_failed", symbol=symbol, error=str(e))
        raise
      finally:
        duration_ms = int((time.perf_counter() - start) * 1000)
        self._log_ingestion(run_id, symbol, status, rows_inserted, errors_count, duration_ms)
    ```

    **Add _sync_to_duckdb(clean, symbol) -> int:**

    - Return count of inserted rows
    - Log with structlog: "duckdb_sync_complete", symbol=symbol, rows=count
    - If DatabaseManager not available (DB not initialized), log warning and skip
