  <action>
    Create `src/swingrl/features/correlation.py` with class `CorrelationPruner`:

    1. `__init__(self, config: SwingRLConfig)`:
       - Store threshold from config.features.correlation_threshold (default 0.85)
       - Define domain-driven drop priority rules:
         ```python
         KEEP_PRIORITY = {
             "rsi_14": 10,          # Prefer RSI over other oscillators
             "macd_histogram": 9,   # Prefer histogram over raw line
             "price_sma50_ratio": 8,
             "price_sma200_ratio": 8,  # Keep both SMAs unless r > 0.90
             "vix_zscore": 7,       # Prefer z-scored over raw
         }
         SMA_EXCEPTION_THRESHOLD = 0.90  # Both SMAs kept unless r exceeds this
         ```

    2. `analyze(self, features: pd.DataFrame) -> pd.DataFrame`:
       - Compute pairwise Pearson correlation: features.corr()
       - Return full correlation matrix

    3. `find_correlated_pairs(self, corr_matrix: pd.DataFrame) -> list[tuple[str, str, float]]`:
       - Find all pairs where abs(r) > threshold
       - Exclude self-correlations
       - Return list of (col_a, col_b, r_value) sorted by abs(r) descending

    4. `select_drops(self, correlated_pairs: list[tuple[str, str, float]]) -> list[str]`:
       - For each correlated pair, apply domain rules:
         - If both are SMA ratios and abs(r) < SMA_EXCEPTION_THRESHOLD: keep both
         - Otherwise: drop the one with lower KEEP_PRIORITY (or lower variance if equal priority)
       - Return list of column names to drop (deduplicated)

    5. `prune(self, features: pd.DataFrame) -> tuple[pd.DataFrame, list[str]]`:
       - Run full pipeline: analyze -> find_correlated_pairs -> select_drops
       - Return (pruned_df, dropped_columns)

    6. `report(self, corr_matrix: pd.DataFrame, dropped: list[str]) -> str`:
       - Generate human-readable summary: which pairs were flagged, which columns dropped, rationale
       - Log with structlog

    This is a one-time pre-training analysis tool, not a runtime component.
  </action>
